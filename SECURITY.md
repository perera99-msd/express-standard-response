# Security Policy

## Supported Versions

Security updates are currently provided for the latest stable release line.

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

Please report security issues privately by email:

msdperera99@gmail.com

Include the following details where possible:

- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested remediation (if available)

## Response Process

- Acknowledgement target: within 72 hours
- Initial assessment target: within 7 days
- Fix timeline: based on severity and complexity

Please do not open public issues for security vulnerabilities.
